## 🚀 How to Use

### **For Users:**
1. Use the app in Hindi (default)
2. See 🌐 हिंदी button at bottom-right
3. Click to switch to English
4. All text changes instantly ✨
5. Click 🌐 English button to switch back

### **For Developers:**
1. Import `t()` from `lib/translations`
2. Use: `{t("key-name", language)}`
3. Get instant Hindi/English support
4. Build & deploy 🎉
